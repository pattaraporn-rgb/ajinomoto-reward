📋 สรุปการแลกของรางวัล

พร้อมส่งให้ลูกค้าแล้ว

นำเข้าข้อมูลจาก Google Sheets
เลือกวิธีที่ง่ายที่สุดสำหรับคุณ
📋 ขั้นตอน (3 คลิก)
1
เปิด Google Sheets → คลิกเมนู File → Download → Comma Separated Values (.csv)
2
กดปุ่ม "เลือกไฟล์ CSV" ด้านขวา แล้วเลือกไฟล์ที่ดาวน์โหลดมา
3
ระบบจะอ่าน คอลัมน์ A = ชื่อ, B = คะแนน และอัปเดตรายการทั้งหมด
💡 ทำแบบนี้ทุกครั้งที่มีของรางวัลใหม่ในชีท — ใช้เวลาไม่เกิน 10 วินาที
รองรับ .csv จาก Google Sheets, Excel
⚠️ ทำไมถึงใช้ไม่ได้บางครั้ง?
• เปิดไฟล์ HTML จากเครื่องโดยตรง (file://) — Browser บล็อก request ออก internet
• ชีทตั้งเป็น "Restricted" หรือ "Anyone in org" เท่านั้น
• ต้องเป็น "Anyone with the link" (public) ถึงจะ Sync ได้
วิธีตั้งค่าชีทให้ Sync ได้:
1. เปิดชีท → คลิก Share (มุมบนขวา)
2. ใต้ "General access" → เปลี่ยนจาก "Restricted" เป็น "Anyone with the link"
3. ตั้งเป็น Viewer → Save
4. Copy link แล้ววางด้านล่าง
